# RCS1028: Format switch section's statement on a separate line

| Property                    | Value      |
| --------------------------- | ---------- |
| Id                          | RCS1028    |

| Default Severity            | Info       |
| Enabled by Default          | \-         |

| Supports Fade\-Out Analyzer | \-         |

## Example

### Code with Diagnostic

```csharp
switch (s)
{
    case "a": return Foo(); // RCS1028
    case "b": return Bar();
}
```

### Code with Fix

```csharp
switch (s)
{
    case "a":
        return Foo();

        return Bar();
}
```
